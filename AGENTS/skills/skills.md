# Skills

## Основные инструменты проекта hh-ru-apply

### Автоматизация hh.ru
- Сбор вакансий по ключевым словам
- Оценка вакансий через LLM (OpenRouter)
- Генерация сопроводительных писем
- Автоматический отклик через браузер (Playwright)

### Навыки (Skills)
- **cover-letter-ats-style**: Правила генерации сопроводительных писем для hh.ru. ATS-оптимизация, избегание признаков ИИ-текста, отзеркаливание требований вакансии.
- **hh-boolean-search**: Полное руководство по поисковым операторам hh.ru для использования в скриптах.
- **vacancy-title-matcher**: Подбор релевантных названий вакансий на основе анализа опыта и компетенций кандидата.

### Команды проекта
- `npm run login` - Вход в hh.ru аккаунт
- `npm run vacancies` - Сбор вакансий
- `npm run harvest` - Оценка вакансий через LLM
- `npm run dashboard` - Запуск веб-дашборда
- `npm run hh-fill-letter` - Отклик через форму вакансии
- `npm run hh-apply-chat` - Отклик через чат с работодателем
- `npm run scan-tg` - Сканирование Telegram-каналов на вакансии

### Кастомные /slash команды проекта
- Нет специальных slash команд проекта, используй стандартные команды npm run

### /-команды Kilo CLI
- /sessions (/resume, /continue) - Переключить сессию
- /new (/clear) - Новая сессия
- /share - Поделиться сессией
- /unshare - Отменить публикацию
- /rename - Переименовать сессию
- /timeline - Перейти к сообщению
- /fork - Разветвить от сообщения
- /compact (/summarize) - Сжать/суммировать сессию
- /undo - Отменить предыдущее сообщение
- /redo - Повторить сообщение
- /copy - Копировать транскрипт сессии
- /export - Экспортировать транскрипт
- /timestamps (/toggle-timestamps) - Показать/скрыть временные метки
- /thinking (/toggle-thinking) - Показать/скрыть блоки мышления
- /models - Переключить модель
- /agents - Переключить агента
- /mcps - Переключить MCP
- /connect - Подключить/добавить провайдера
- /status - Посмотреть статус
- /themes - Переключить тему
- /help - Показать помощь
- /editor - Открыть внешний редактор
- /exit (/quit, /q) - Выйти
- /profile (/me, /whoami) - Посмотреть профиль Kilo Gateway
- /teams (/team, /org, /orgs) - Переключить между командами Kilo Gateway
- /remote - Переключить режим удалённого доступа
- /init - Создать/обновить AGENTS.md для проекта
- /local-review - Рецензировать изменения ветки
- /local-review-uncommitted - Рецензировать некоммитнутые изменения

### Технический стек
- Node.js + Playwright для браузерной автоматизации
- OpenRouter API для LLM (модели Qwen)
- Docker для контейнеризации
- Express.js для веб-дашборда
