Варианты UI — кнопка "Обновить"

Все варианты показывают idle / loading / done состояния. Нажмите кнопки под демо чтобы переключать.

Вариант 1 — Progress Ring
12 вакансий для обновления
Круглая кнопка 72×72px с вращающимся кольцевым индикатором. Под кнопкой — счётчик оставшихся вакансий.
Вариант 2 — Fullwidth Bar
Кнопка на всю ширину контейнера. Прогресс-бар окрашивает фон слева направо. После завершения — галочка на 2 сек.
Вариант 3 — Pill with Orbital
8 из 20 обновлено
Кнопка-капсула ~180×56px. Две точки-орбиты вращаются вокруг кнопки во время загрузки. Под кнопкой — облачко статуса.
