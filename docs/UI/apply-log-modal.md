# Лог отклика — модалка

## Описание

Модальное окно для просмотра лога выполнения `hh-apply-chat`.

**Селектор:** `#apply-log-modal`
**Файл:** `dashboard/public/index.html` (строки 164-180)

---

## Структура

```
Лог «Отклик в браузере»
Файл на диске: data/hh-apply-chat.log
В терминале: tail -f data/hh-apply-chat.log

[Обновить]

┌─────────────────────────────────────────────────────────────┐
│ [текст лога]                                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## Элементы

| Элемент | Описание |
|---------|----------|
| `.apply-log-hint code` | Путь к файлу лога |
| `.btn-refresh-apply-log` | Кнопка "Обновить" |
| `.apply-log-pre` | Pre с текстом лога |

---

## Автообновление

Лог обновляется автоматически каждые 2.5 секунды через `setInterval` в `refreshApplyLogModal()`.

При закрытии модалки интервал очищается.

---

## Файл лога

По умолчанию: `data/hh-apply-chat.log`

---

## Состояния

| Состояние | Текст в pre |
|-----------|-------------|
| Файла нет | "Файла лога ещё нет. Нажмите «Отклик в браузере»..." |
| Ошибка | "Ошибка: {message}" |
| Загрузка | "Загрузка…" |
| Есть лог | Текст лога |
