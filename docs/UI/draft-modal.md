# Модалка "Черновик письма"

## Описание

Модальное окно для редактирования и утверждения сопроводительного письма.

**Селектор:** `#draft-modal`
**Файл:** `dashboard/public/index.html` (строки 153-163), `dashboard/public/app.js` (функция `openDraftModal`)

---

## Структура

```
┌─────────────────────────────────────────────────────────────┐
│ ×                                                         │
│                                                             │
│ Черновик письма                                            │
│ Операционный директор (COO)                                 │
│                                                             │
│ Вариант                                                    │
│ ○ Вариант 1   ○ Вариант 2   ○ Вариант 3                   │
│                                                             │
│ Текст (правки сохраняются и учитываются при следующей      │
│ генерации)                                                  │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │                                                         │ │
│ │                                                         │ │
│ │                                                         │ │
│ │                                                         │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                             │
│ [Сохранить правки]  [Утвердить]  [Отклонить]                │
└─────────────────────────────────────────────────────────────┘
```

---

## Элементы

| Элемент | Описание |
|---------|----------|
| `.modal-close` | Кнопка закрытия × |
| `.modal-title` | "Черновик письма" |
| `.modal-vacancy` | Название вакансии |
| `.modal-draft-fieldset` | Fieldset с радио-кнопками |
| `.modal-letter-label` | Подсказка под textarea |
| `.modal-letter-edit` | Textarea rows=12 |
| `.modal-draft-actions` | Ряд кнопок |

---

## Состояния

| Состояние | UI |
|-----------|-----|
| **Нет вариантов** | "Нет вариантов." в body модалки |
| **Есть варианты** | Радио + textarea + кнопки |
| **Сохранение** | Кнопки disabled |
| **Ошибка** | alert с сообщением |

---

## Действия

| Действие | Кнопка | API | Результат |
|----------|--------|-----|-----------|
| Переключить вариант | Радио "Вариант 1/2/3" | — | Меняет textarea |
| Сохранить правки | "Сохранить правки" | `POST /api/cover-letter/save-draft` | toast "Правки сохранены" |
| Утвердить | "Утвердить" | `POST /api/cover-letter/action` action=approve | toast "Письмо утверждено", закрывает модалку |
| Отклонить | "Отклонить" | `POST /api/cover-letter/action` action=decline | toast "Черновик отклонён", закрывает модалку |

---

## Keyboard

| Клавиша | Действие |
|---------|----------|
| Escape | Закрыть модалку |
| Enter (в textarea) | Не отправляет форму (type=button) |
