# Генерация сопроводительных писем

## Обзор

AI cascade генерирует 3 варианта письма на основе данных вакансии и стиля пользователя.

---

## Файлы

| Файл | Назначение |
|------|------------|
| `lib/cover-letter-openrouter.mjs` | Основная логика генерации |
| `lib/cover-letter-style-context.mjs` | Контекст стиля |
| `config/cover-letter.txt` | Шаблон |
| `config/cover-letter-style-examples.txt` | Примеры писем пользователя |

---

## Процесс

### 1. Запуск генерации

**Запуск возможен двумя способами:**
- Вручную: кнопка "Сгенерировать письмо" в блоке "Вариант" (вкладка "Вручную")
- Автоматически: сразу после успешного скрапинга вакансии (вкладка "Вручную", действия "Загрузить" и "➕ Добавить")

**API:** `POST /api/cover-letter/generate`

**Тело:**
```json
{ "id": "vacancy-uuid", "force": false }
```

---

### 2. AI Cascade

```
OpenRouter (free models) → Google Gemini → Groq
```

**Промпт:**
- Данные вакансии (title, company, description)
- CV пользователя (из папки `CV/`)
- style-examples (предыдущие письма пользователя)
- Шаблон cover-letter.txt

**Результат:** 3 варианта письма

---

### 3. Сохранение

**API:** `POST /api/cover-letter/save-draft`

**Тело:**
```json
{
  "id": "vacancy-uuid",
  "variants": ["Вариант 1...", "Вариант 2...", "Вариант 3..."]
}
```

**Результат:** `coverLetter.variants = [...]`, `coverLetter.status = "pending"`

---

### 4. Редактирование

**Модалка:** "Черновик письма" (`#draft-modal`)

- Выбор варианта (радио)
- Редактирование в textarea
- Сохранение промежуточных правок

---

### 5. Утверждение

**Кнопка:** "Утвердить"

**API:** `POST /api/cover-letter/action` action=approve

**Тело:**
```json
{
  "id": "vacancy-uuid",
  "action": "approve",
  "text": "утверждённый текст письма"
}
```

**Результат:**
- `coverLetter.status = "approved"`
- `coverLetter.approvedText = text`

---

## Повторная генерация

Если нужно пересоздать варианты:
- Кнопка "Снова сгенерировать письмо" (для declined)
- `POST /api/cover-letter/generate` с `force: true`

---

## Контекст стиля

Берётся из:
1. `config/cover-letter-style-examples.txt` — примеры писем
2. `lib/cover-letter-style-context.mjs` — формирует промпт

---

## Обработка ошибок

| Ошибка | Что делать |
|--------|------------|
| AI не ответил | Попробовать снова или использовать последний черновик |
| Текст пустой | Нельзя утвердить — требуется ввод текста |
| Превышен лимит | Подождать, попробовать с другим провайдером |
