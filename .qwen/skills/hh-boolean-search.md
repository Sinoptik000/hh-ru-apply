# HH.ru Boolean Search

Полное руководство по поисковым операторам hh.ru для использования в скриптах.

## Поведение по умолчанию

- **Синонимы:** поиск автоматически добавляет синонимы (`директор` → `управляющий`, `руководитель`)
- **Нормализация:** слова приводятся к одной форме (склонения, спряжения)

## Операторы

### Точное совпадение

| Оператор | Синтаксис | Назначение |
|----------|-----------|------------|
| `!` | `!слово` или `!"фраза"` | Отключает синонимы и нормализацию |
| `""` | `"слово1 слово2"` | Строгий порядок слов рядом |

### Близость и префиксы

| Оператор | Синтаксис | Назначение |
|----------|-----------|------------|
| `~N` | `"фраза"~N` | Слова на расстоянии ≤N друг от друга |
| `*` | `префикс*` | Поиск по префиксу слова |

### Булевы операторы (приоритет по убыванию)

| Оператор | Приоритет | Назначение |
|----------|-----------|------------|
| `NOT` | Высший | Исключает слово |
| `AND` | Средний | Все слова должны быть (пробел = неявный AND) |
| `OR` | Низший | Хотя бы одно из слов |
| `( )` | - | Группировка приоритета |

### Поиск по полям

**Синтаксис:** `ПОЛЕ:значение` (без пробела после `:`)

**Вакансии:** `NAME`, `COMPANY_NAME`, `DESCRIPTION`

**Резюме:** `POSITION`, `EDUCATION`, `KEYWORDS`, `CONTACT_INFO`, `WORKPLACE_ORGANIZATION`, `WORKPLACE_POSITION`, `WORKPLACE_DESCRIPTION`

### Точное равенство поля

**Синтаксис:** `^ПОЛЕ:значение` (полное совпадение, не содержит)

Применимо к: `NAME`, `COMPANY_NAME` (вакансии), `POSITION`, `EDUCATION`, `WORKPLACE_ORGANIZATION`, `WORKPLACE_POSITION` (резюме)

### Специальные

| Синтаксис | Назначение |
|-----------|------------|
| `!id:(12345)` | Поиск по ID вакансии |
| `!manager_id:(12345)` | Вакансии конкретного менеджера |

## Правила

1. ❌ **Запрещены пробелы** после `!`, `~`, `:`
2. ✅ Операторы регистронезависимы, но рекомендуется `NOT`, `AND`, `OR`
3. ✅ Приоритет: `NOT` > `AND` > `OR`
4. ✅ Можно многократно вкладывать: `(A OR B) AND NOT (C OR D)`

## Примеры

```
# Точное название должности
!"операционный менеджер"

# Исключение слова
!"операционный менеджер" NOT удаленно

# Комбинация синонимов
!("операционный менеджер" OR "operations manager" OR "менеджер по операциям")

# С префиксом
!гео*

# Близость слов
!"менеджер продажи"~5

# По полю NAME
NAME:!"операционный менеджер"

# Комплексный запрос
^NAME:!"операционный менеджер" AND (Москва OR "Московская область") NOT стажер

# По нескольям полям резюме
POSITION:!"финансовый директор" AND NOT !заместитель EDUCATION:"Финансовый университет"
```

## Ограничения

- GUI «Расширенный поиск» НЕ поддерживает вложенные запросы
- Оператор `!` и галочка «Точная фраза» в GUI — взаимоисключающие
- Пустой запрос запускает ML-подборку без ключевых слов
