# hh-ru-apply

Минимальный каркас на Node.js и Playwright для работы с сайтом [hh.ru](https://hh.ru) в отдельном браузерном профиле (логин один раз, дальше тот же профиль).

Проект **не связан** с другими репозиториями; здесь только скрипты под hh.ru.

## Важно

- Автоматизация откликов и массовые действия могут **противоречить правилам** сервиса и привести к ограничению аккаунта.
- Возможны **капча** и другие меры защиты; рассчитывайте на **ручной вход** и умеренные паузы между шагами.
- Скрипт `apply.mjs` сейчас только **проверяет**, что сохранённая сессия жива. Логику отклика к конкретным вакансиям нужно дописывать под актуальную вёрстку hh.ru.

## Требования

- Node.js 18+
- После `npm install` при необходимости: `npx playwright install chromium`

## Настройка

```bash
cp .env.example .env
```

При необходимости задайте `HH_SESSION_DIR` в `.env`.

## Первый вход

```bash
npm run login
```

В открывшемся окне войдите на hh.ru, затем нажмите Enter в терминале. Профиль сохраняется в `data/session/chromium-profile` (или в `HH_SESSION_DIR`).

## Проверка сессии

```bash
npm run apply
```

Опционально с открытым окном для отладки:

```bash
node scripts/apply.mjs --stay-open
```

## Шаблон сопроводительного

Пример текста: [config/cover-letter.example.txt](config/cover-letter.example.txt). Скопируйте в свой файл (например `config/cover-letter.txt`) и добавьте `config/cover-letter.txt` в `.gitignore`, если не хотите коммитить личные данные.

## Структура

- `scripts/login.mjs` — сохранение сессии.
- `scripts/apply.mjs` — проверка сессии (заглушка под будущие сценарии отклика).
